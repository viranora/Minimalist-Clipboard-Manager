Clarity Clip - Minimalist Clipboard Manager

A sleek, simple, and visually appealing clipboard history manager built with Electron. This desktop widget runs quietly in the background, tracks your clipboard history, and makes it easy to retrieve previously copied items.

This project was built to explore Electron's native capabilities (like the clipboard API) and create a useful desktop utility that also looks beautiful, featuring animated gradients and a "glassmorphism" effect.

Clipboard History: Automatically saves the last 10 text items you've copied.

Animated UI: A vibrant, animated gradient background (lila/pink tones) makes the app visually engaging.

Glassmorphism List: History items use a semi-transparent, blurred background effect.

Instant Feedback: Get a "Copied!" notification when you re-copy an item.

Minimalist: Fixed-size (350x500) window designed to be a simple, focused utility.
